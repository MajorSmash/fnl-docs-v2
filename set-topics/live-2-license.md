Users with an existing FluidNinja **LIVE-1** license could apply for FluidNinja LIVE-2 BETA.
License is validated via "EPIC Order ID" : a number on your purchase-receipt.

This post explains **how to access your EPIC Order ID**:

1. Visit this URL: https://www.epicgames.com/account/transactions ---- This takes you to the "Transactions" menu.
2. Under "Transactions" select "Purchase" - you'll get a list of all items ever purchased from EPIC
3. Look up "FluidNinja LIVE" ---- Click on the Product name, shown in blue. A small info-field rolls down, showing "**Order ID**"

Order ID is a **17 DIGIT** numbers+letters sequence.
Optionally, there is an "email receipt" button in the info-field. This button emails the receipt as PDF (with the Order ID).
