The plan: yes, I would like to port ninja to UE6 - once UE6 is available.
- UE6 preview is scheduled for late 2027 / early 2028 --- so it might be a bit early to start evaluating this 😆
- The architectural changes intruduced by UE6 are not fully known. For example: ninja LIVE-2 core is running on niagara - but the asset mangement is done by wrapper blueprints. In case blueprints are deprecated under UE6 (as it has been announced): we might have to completely refactor ninja workings --- there is a great amount of uncertainity here - again, a bit too early to formulate an opinion
